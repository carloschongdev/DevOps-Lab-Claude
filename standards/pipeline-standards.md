# CI/CD Pipeline Standards

Pipelines in this laboratory must simulate real engineering workflows.

---

## Pipeline Stages

Typical pipeline stages include:

1. build
2. test
3. artifact generation
4. container build
5. deployment

---

## Required Behaviors

Pipelines must:

- fail on errors
- generate logs
- store artifacts
- provide clear stage output

---

## Debugging

Students must inspect pipeline logs when failures occur.
