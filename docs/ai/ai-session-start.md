# AI Session Start

When starting a new AI session for the DevOps Lab, follow this initialization procedure.

1. Review the project documentation using the reading order defined in:

ai-reading-order.md

2. Identify the current lab state using:

ai-mentor-state.md
lab-progress.md

3. Review the rules for generating instructions:

ai-operating-rules.md
ai-task-generation-rules.md

4. Ensure that the next instructions follow the phase progression defined in:

lab-phase-map.md

After reviewing the documentation, generate the next DevOps Lab tasks following the execution protocol defined in:

lab-execution-protocol.md
