# DevOps Lab Phase Map

This document describes the major phases of the DevOps Lab.

The purpose is to ensure that the AI mentor generates instructions
that follow the correct progression of the lab.

---

# Phase 1 — Environment Setup

Development environment preparation.

Topics:

• Git workflow
• repository setup
• development tooling

---

# Phase 2 — Docker Fundamentals

Core container concepts.

Topics:

• Dockerfile
• image building
• container execution
• container debugging

---

# Phase 3 — Containerized Application

Running applications inside containers.

Topics:

• application packaging
• container runtime behavior
• health checks

---

# Phase 4 — CI/CD Foundations

Introduction of pipeline automation.

Topics:

• local pipeline execution
• GitHub Actions CI
• container build automation
• artifact generation

---

# Phase 5 — Infrastructure as Code

Automating infrastructure provisioning.

Topics:

• Terraform
• infrastructure modules
• environment configuration

---

# Phase 6 — Kubernetes

Container orchestration.

Topics:

• Kubernetes architecture
• deployments
• services
• scaling

---

# Phase 7 — Observability

Monitoring and logging.

Topics:

• metrics collection
• Prometheus
• Grafana
• log aggregation

---

# Phase 8 — DevOps Automation

Automation and tooling.

Topics:

• Python automation
• pipeline improvements
• DevOps tooling

---

# Phase 9 — Production Simulation

Final lab phase.

Topics:

• production-style architecture
• deployment pipelines
• reliability practices
